Instructions for installing the Contact Form 7 integration plugin for WordPress and SuiteCRM

Contact Form 7 integration plugin with SuiteCRM. Requests are automatically entered in SuiteCRM under the sections Leads, Contacts, Counterparties, and Appeals. You can create as many forms as you wish with any data types available in SuiteCRM API and attach files. Easy synchronization of custom/custom fields between SuiteCRM and the plugin.

Features

Easy customization of integration;

Unlimited number of forms;

Enabling/Disabling of individual Contact Forms 7;

Choice of module type - Leads, Contacts, Accounts, Cases;

Selection of the person responsible for new requests;

Configuring the correlation of Contact Form 7 tags with SuiteCRM fields;

Possibility to automatically attach files that will be displayed in Notes and History (Leads, Contacts, Counterparties or Appeals).

Enabling/Disabling error logging. In the root of the plugin's directory in the logs folder you will create dated log files, like suitecrm-log-<dd-mm-YYYYY>.txt

Enable/disable to delete transferred files on the site side, which are saved in wp-content/uploads/suitecrm_uploads (by default files are not saved);

Module types, responsible persons, and SuiteCRM fields are sorted alphabetically in ASC order ([A-Z]);

 

Installing the plugin

To start working with the plugin, follow these steps:

1.       Download the desired plugin as a cf7-integration-with-suitecrm.zip file.

2.       From your WordPress dashboard, choose Plugins > Add New.

3.       Click Upload Plugin at the top of the page.

4.       Click Choose File, locate the plugin cf7-integration-with-suitecrm.zip file, then click Install Now.

5.       After the installation is complete, click Activate Plugin.

To add a plugin to your WordPress website via FTP:

1. Download the desired plugin as a cf7-integration-with-suitecrm.zip.

2. Extract (or unzip) the plugin file. The unzipped folder will appear with the same name as the .zip file.

3. Next, you’ll need to connect to your WordPress website’s server with FTP. To do this, you can use an FTP client that connects to the host server to transfer files.

4. Once you open the FTP client, connect to your website with a username and password. You might need to obtain these credentials from your hosting provider.

5. In your WordPress file directory, open the wp-content folder, then open the plugins folder.

6. Upload the unzipped plugin folder that you downloaded to the plugins folder.

7. Return to your WordPress dashboard and select Plugins > Installed Plugins. Find your new plugin and click Activate.

 

In the Contact form 7 drop-down menu, select the SuiteCRM Integration tab and fill in all the fields:

·         SuiteCRM URL: - e.g. http://mycrm.com/service/v4_1/rest.php

·         SuiteCRM Admin User: - SuiteCRM user login with administrator rights

·         SuiteCRM Admin Password: - password for this administrator account

You can use the Test Connection button to verify that the data you entered is correct, or you can save the settings by clicking Save Changes.

To upload or synchronize fields with SuiteCRM, click Synchronize Fields.

Form Setup

Create a form in the Contact form 7 plugin, save it. After saving the form, a new SuiteCRM tab appears.

This tab contains a list of available parameters:

Enable/Disable form - Enable/Disable form and data transfer in SuiteCRM.

Module type - drop-down list for selecting a module: Lead, Contact, Counterparty or Conversation.

Choose a person in charge - a drop-down list of SuiteCRM users with which you can specify who is responsible for new requests.

Settings contact form tags and SuiteCRM field types - a table of relations between form tags and SuiteCRM fields.

To pass internal information in SuiteCRM, such as Job Title, Lead Source, Lead Source Description, etc., you can use the tag construction

[hidden lead_source "lead_source_name"]

Where:

hidden - the designation of the hidden Contact form 7 tag

lead_source - the exact name of the field in SuiteCRM

"lead_source_name" - the name of the source in SuiteCRM

On the SuiteCRM tab you must specify the tag's correspondence to the field
